# Coca-Cola-Sales-Dashboard
## Objective
Build an interactive sales dashboard to help managers quickly understand how Coca-Cola products are performing across **sales, operating profit, pricing, and regions**.  This dashboard is designed to make it easy to **spot trends, compare performance by state and brand, and focus on what drives profit**, all in one place.

## Key Insights
### Sales & Regional Performance
-Visualize sales distribution by **state and region** using an interactive map to identify growth opportunities and underperforming markets.
### Operating Profit & Margin Drivers
-Use the **Key Influencers** visual to understand the key factors impacting operating profit and profit margins across Coca-Cola brands.
### Financial & Product Analysis
-Analyze financial metrics such as **sales, price per unit, and operating profit** through a matrix view to compare product and brand performance.
### Trend & Time-Based Analysis
-Track changes in sales and profitability over time using a **dynamic date filter**, supporting clear trend analysis.
### AI-Powered Business Questions
-Explore insights instantly using the **Power BI Q&A visual**, allowing users to ask natural language questions and get quick answers.

## Tools & Technologies
- Power BI Desktop  
- Power Query (Data extraction & transformation from Excel)  
- DAX (Measures and calculations)  
- Data Modeling  
- Power BI visuals:
  - AI-powered Q&A Visual  
  - Key Influencers (Operating Profit analysis)  
  - Map Chart (Sales by State/Region)  
  - Matrix (Financial & pricing breakdown)  
  - Date Slicer / Timeline (Interactive filtering)
